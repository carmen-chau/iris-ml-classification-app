About the Application

Project Inspiration:

The inspiration for this project came about when I was first introduced to Artificial Intelligence (AI) and Machine Learning (ML) at my first hackathon last summer. I became drawn to the analytical and predictive powers ML had especially with large groups of data, and so I became drawn to experiment with ML concepts.

I have also always had an interests in flowers, specifically in the Iris species due to their bright colours and symmetrical shapes

So, I decided to combine my interests in both ML and in botanics by creating an Iris ML classification web application developed and trained using the University of California Irvine (UCI) Iris Data Set.

Languages and Frameworks used:
ML Script: Python, Flask
Front end Application:HTML, CSS, BootStrap, JavaScript
